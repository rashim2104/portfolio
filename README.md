# **Rashim R B's Portfolio**

A personal portfolio website showcasing my projects and achievements.

# **Getting started**

Magic Portfolio was built with [Once UI](https://once-ui.com) for [Next.js](https://nextjs.org). It requires Node.js v18.17+.

**1. Clone the repository**
```
git clone https://github.com/once-ui-system/magic-portfolio.git
```

**2. Install dependencies**
```
npm install
```

**3. Run dev server**
```
npm run dev
```

**4. Edit config**
```
src/app/resources/config
```

**5. Edit content**
```
src/app/resources/content
```

**6. Create blog posts / projects**
```
Add a new .mdx file to src/app/blog/posts or src/app/work/projects
```

# **Features**

## **Core Features**
- Responsive layout optimized for all screen sizes
- SEO optimization with automatic metadata generation
- Project showcase with detailed case studies
- Blog section for technical writing
- Gallery for achievements and event highlights

## **Technical Stack**
- Built with Next.js and React
- TypeScript for type safety
- Modern UI components
- MDX for content management
- Automatic SEO optimization

- Project showcase with MDX
