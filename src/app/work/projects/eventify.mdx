# Eventify

## Overview
A collaborative project building a robust event management system for Sairam Institutions, handling multiple monthly events with comprehensive venue and attendee management features.

## Key Features
- Venue management system
- Attendee tracking and management
- User authentication and authorization
- Event scheduling system
- Real-time notifications
- Automated reporting

## Technical Stack
- Next.js for frontend and API routes
- MongoDB for database management
- AWS S3 for media storage
- AWS EC2 for deployment

## Impact
- Successfully managing 10+ events monthly
- Streamlined event organization process
- Enhanced attendee experience
- Continuous improvements based on user feedback
