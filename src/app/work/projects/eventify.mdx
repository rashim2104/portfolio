# Eventify

## Overview
An event management platform I built with a senior for Sairam Institutions. It handles the college's events end to end — venues, attendees, scheduling — and they still run 10+ events a month on it.

## Features
- Venue management
- Attendee tracking
- Auth and roles
- Event scheduling
- Real-time notifications
- Automated reporting

## Stack
- Next.js for frontend and API routes
- MongoDB
- AWS S3 for media
- AWS EC2 for deployment

## Where it stands
Still in active use, running 10+ events a month. We kept fixing and improving it as the college used it and told us what broke.
