# Adutha Ilakku Ticketing System

## Overview
A QR-based ticketing and attendee system for a large college event. Scan a code at the gate, the person's verified, and the dashboard updates live.

## Features
- QR scanning for attendee verification at entry
- Live attendance dashboard
- Kit delivery tracking
- Handled 5,000+ attendees
- Reporting on who's in and what's been handed out

## How it worked
Each attendee got a dynamically generated QR code. Scanning it at the gate verified them in real time, marked their kit delivery status, and fed the analytics dashboard so organizers always knew the headcount.
