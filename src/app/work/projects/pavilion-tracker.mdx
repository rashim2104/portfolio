# QR-Based Pavilion Tracking System

A sophisticated tracking system designed to monitor real-time occupancy across multiple pavilions for an event with over 11,400 attendees spanning 4 days.

## Key Features

- Real-time pavilion occupancy tracking
- Live occupancy charts and visualizations
- Crowd management assistance
- Multi-pavilion monitoring
- QR-based check-in/check-out system

## Technologies Used

- Next.js for the full-stack application
- MongoDB for real-time data storage
- WebSocket for live updates
- Data visualization for occupancy charts
- QR code generation and scanning

## Impact

- Successfully managed crowd flow for 11,400+ attendees
- Enabled real-time monitoring across multiple pavilions
- Helped crowd management teams optimize attendee distribution
- Minimized congestion through data-driven decisions
